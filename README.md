# Nigeria-Sales-Dashboard-PowerBI
Interactive Sales Dashboard built using Power BI. Performed data cleaning, handled missing values, created DAX measures, and developed interactive visualizations to analyze sales performance across products, states, channels, and time.
Project Overview

This project presents an interactive Sales Performance Dashboard built using Power BI. The objective was to clean a messy sales dataset, create meaningful KPIs, and provide actionable business insights through interactive visualizations.

Business Problem

Organizations need a clear understanding of their sales performance across products, states, sales channels, and time periods. This dashboard helps stakeholders monitor key metrics and identify trends for data-driven decision-making.

Dataset Information

The dataset contains:

Customer Name
State
Product
Units Sold
Unit Price
Total Sale
Sale Date
Sales Channel
Order ID

The dataset included missing values, inconsistent text formatting, and incomplete records which were cleaned using Power Query.
